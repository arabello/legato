# Legato

A web-based DJ mix planning tool focused on **harmonic mixing**. Create harmonically cohesive DJ sets by visualizing key compatibility and planning smooth transitions between tracks.

**Live Demo:** https://matteopellegrino.dev/legato

**Based on:** https://blog.native-instruments.com/harmonic-mixing-rules-and-how-to-break-them/

## Tech Stack

- React 19 + React Router 7
- TypeScript
- Tailwind CSS 4
- Vite 7
